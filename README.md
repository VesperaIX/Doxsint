<img src="https://github.com/ZeroSaturnn/Doxsint/raw/main/src/video.mkv"><br>

# Instalation
1. `$ git clone https://github.com/ZeroSaturnn/Doxsint && cd Doxsint && chmod +x start.sh`
2. `./start.sh`

# Alternative Mode
`$ ./start.sh -u (username) -s (site [optional])`.
